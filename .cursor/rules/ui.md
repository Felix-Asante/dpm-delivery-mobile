# UI library

Use HeroUI Native documentation from:
https://v3.heroui.com/native/llms-full.txt

Always import UI components from HeroUI Native
Prefer HeroUI Native components instead of plain React Native primitives

# Cursor rules — styling

Use NativeWind (Tailwind) classes for styling React Native components
Do not use inline styles if an equivalent Tailwind utility exists
Prefer utility-first approach from UniWind documentation: https://docs.uniwind.dev/llms-full.txt
